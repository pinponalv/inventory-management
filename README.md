# Inventory-management

Sistema de gestion de inventarios que permite administrar nuestros productos, proveedores, almacenes, poder registrar movimientos de entrada de manera automatica y manual, generar compras

## Technologies
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Docker compose

## Features
- CRUD Products
- CRUD Supplier
- CRUD Warehouse
- CRUD PurchaseOrders
- CRUD InventoryMovements
- CRUD Iventory
- CRUD User

## Future Improvements
- Spring Security
- JWT Authentication
- Role-based access control
- Swagger/OpenAPI Documentation
- Unit and Integration Testing
